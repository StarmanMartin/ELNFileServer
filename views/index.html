How to setup!

  1. Download Installer

    mkdir ~/eln_installer
    cd ~/eln_installer
    wget https://github.com/StarmanMartin/ELNFileServer/releases/download/latest/installer

  2. Run Installer

    sudo chmod +x installer
    sudo su -c ./installer

  3. Follow instructions

How add instance!

  1. Run Installer

    cd ~/eln_installer
    sudo su -c ./installer

  2. Follow instructions

How update all instance!

  1. Run Installer with tag

    cd ~/eln_installer
    sudo su -c "./installer -u"

  2. Follow instructions

Hints

During the process you will be asked to enter the RSA public key.
  1. Generate an SSH Key Pair on your local mashine. If you use linux run

    ssh-keygen -b 2048 -t rsa

  2. Copy the content of the public key (id_rsa.pub) into the installer terminal

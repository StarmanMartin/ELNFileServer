How to setup!

  1. Download Installer

    mkdir ~/eln_installer
    cd ~/eln_installer
    wget https://github.com/StarmanMartin/ELNFileServer/releases/tag/latest/installer

  2. Run Installer

    sudo chmod +x installer
    sudo su -c ./installer

  3. Follow instructions

How add instance!

  1. Run Installer

    sudo su -c ./installer

  2. Follow instructions

How update all instance!

  1. Run Installer with tag

    sudo su -c "./installer -u"

  2. Follow instructions
